## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## Usage

1. Check off each habit as you complete it throughout the day
2. Watch your progress circle fill up as you complete more habits
3. View your progress over the last 30 days in the chart
4. Track your average completion rate and current streak

## Technologies

- React 18
- Vite
- Recharts (for graph visualization)
- date-fns (for date handling)
